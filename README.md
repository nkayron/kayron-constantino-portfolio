# Portfólio Pessoal — Kayron Constantino

Site estático de portfólio pessoal construído com HTML5, CSS3 e JavaScript puro.

## Arquivos incluídos
- `index.html` (página inicial)
- `sobre.html` (Sobre mim)
- `formacao.html` (Formação)
- `portfolio.html` (Portfólio)
- `contato.html` (Contato — formulário com validação)
- `style.css` (estilos, modo claro/escuro)
- `script.js` (validação do formulário, modo claro/escuro, menu responsivo)

## Deploy automático (GitHub Pages)

1. Crie um repositório no GitHub e envie todos os arquivos para a branch `main`.
2. No repositório, vá em **Settings → Pages** e selecione publicar a partir da branch `main` (root).
3. Após alguns minutos, o site ficará disponível no endereço fornecido pelo GitHub Pages.

## Observações

- O formulário é apenas front-end: a validação e o alerta são locais.
- Para enviar mensagens para um servidor/serviço real, integre um backend ou serviço de terceiros (Formspree, Netlify Forms, etc.).
